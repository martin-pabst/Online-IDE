Java-Online: Keyboard-Shortcuts
Allgemein
F1 Öffnet die Eingabeleiste für Editor-Befehle
Strg + Mausrad Editor-Zoom
F11 Vollbild ein/aus
Programm starten/debuggen
F4 Programm starten/pausieren
F6 Einzelschritt (Step over)
F7 Einzelschritt (Step into)
Grundlegendes Editieren
Strg + X Ausschneiden (ganze Zeile, falls nichts markiert)
Strg + C In die Zwischenablage kopieren (ganze Zeile, falls nichts markiert)
Alt + Pfeil oben/unten Den markierten Bereich (bzw. die aktuelle Zeile, falls nichts markiert) nach oben/unten verschieben
Shift + Alt + Pfeil oben/unten Den markierten Bereich (bzw. die aktuelle Zeile, falls nichts markiert) nach oben/unten verdoppeln
Strg + Shift + K Die ganze Zeile löschen
Strg + Enter Eine Zeile unterhalb des Cursors einfügen
Strg + Shift + Enter Eine Zeile oberhalb des Cursors einfügen
Strg + K Finde die korrespondierende Klammer
Pos1 / Ende Gehe an den Beginn/ans Ende der Zeile
Strg + Pos1 / Strg + Ende Gehe an den Beginn/ans Ende der Datei
Strg + Pfeil auf/ab Scrolle rauf/runter ohne die Cursorposition zu ändern
Alt + Bild auf/Bild ab Blättere eine Seite rauf/runter ohne die Cursorposition zu ändern
Strg + # und Strg + , Kommentiere den markierten Bereich/die Zeile ein/aus
Shift + Alt + A Blockkommentar ein/aus
Strg + G Gehe zu Zeile...
Suchen/Ersetzen
Strg + F Suchen ("find")
Strg + H Suchen und Ersetzen
F3/Shift + F3 Finde nächstes/vorhergehendes
Alt + Enter Markiere alle Fundstellen und setze je einen Cursor rein
Arbeiten mit mehreren Cursorn gleichzeitig
Strg + D Markiere etwas -> Strg + D findet das nächste Vorkommen und setzt einen zusätzlichen Cursor rein
Alt + Click Füge einen weiteren Cursor ein
Strg + Alt + Pfeil rauf/runter Füge einen weiteren Cursor eine Zeile drüber/drunter ein
Strg + U Mache die letzte Cursoraktion rückgängig
Shift + Alt + I Füge je einen Cursor am Ende jeder markierten Zeile ein
Wort selektieren, dann Strg + Shift + L Selektiere jedes Vorkommen dieses Wortes im ganzen Text und füge je einen Cursor ein
Strg + F2 Selektiere jedes Vorkommen des aktuellen Wortes im ganzen Text und füge je einen Cursor ein
Text selektieren
Shift + Pfeiltasten Selektiere zeichenweise
Shift + Strg + Pfeiltasten Selektiere wortweise
Strg + L Selektiere die aktuelle Zeile
Shift + Alt + Pfeil rechts/links Aktuelle Selektion intelligent erweitern/verkleinern
Shift + Alt + Mit der Maus selektieren Selektion eines rechteckigen Textbereiches
Strg + Shift + Alt + Pfeiltasten Selektion eines rechteckigen Textbereiches
Programmierhilfen
Strg + Leertaste Vorschläge machen (kontextsensitive Hilfe)
Strg + Shift + Leertaste Parametervorschläge machen
Shift + Alt + F Programm schön formatieren
F12 Geh zur Definition des Symbols, in dem der Cursor gerade steht
Alt + F12 Öffne kleines "Peek-Fenster", in dem die Definition des aktuellen Symbols gezeigt wird
F2 Benenne das Symbol um, in dem der Cursor steht und berücksichtige dabei alle weiteren Vorkommen des Symbols.
